# TASK 3 - Recreate a Simple UI from Figma

Recreate a Simple UI from Figma application built with Next.js, TypeScript, and modern web technologies.

## Features

- **Responsive Design**: Mobile-first design that works on all devices
- **Type-Safe**: Full TypeScript implementation with Zod validation
- **Data Fetching**: Static array

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui
- **Animation**: framer-motion
- **Data Fetching**: Static array

## Getting Started

### Prerequisites

- Node.js 24+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Kerolos2000/recreate-a-simple-ui-from-figma
```

2. Install dependencies:
```bash
npm install
# or
yarn
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
